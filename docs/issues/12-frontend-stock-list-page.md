## 개요

React 프론트엔드에서 거래량 상위 200개 종목을 검색, 필터, 정렬, 페이지네이션으로 조회하는 종목 목록 화면을 구현한다.

## 작업 범위

- [ ] 종목 목록 라우팅 페이지를 구성한다.
- [ ] 종목 목록 조회 API를 호출한다.
- [ ] 종목명 또는 종목코드 검색 입력을 제공한다.
- [ ] 시장 구분 필터를 제공한다.
- [ ] 최신 종가, 등락률, 거래량 정렬을 제공한다.
- [ ] 페이지네이션을 제공한다.
- [ ] 종목코드, 종목명, 시장 구분, 최신 종가, 등락률, 거래량, 마지막 거래일을 표시한다.
- [ ] 로딩, 빈 상태, 오류 상태를 화면 영역별로 표시한다.

## 검증 흐름

- [ ] 종목 목록 화면에서 검색어, 시장 필터, 정렬, 페이지를 변경했을 때 API 요청 조건과 목록 표시가 함께 갱신되는지 테스트한다.

## 제외 범위

- 시장 개요 화면은 제외한다.
- 종목 상세 화면은 제외한다.
- Spring Boot API 구현은 제외한다.
- Python 배치 구현은 제외한다.

## 완료 기준

- [ ] 구현 범위가 이슈 목적을 벗어나지 않는다.
- [ ] 관련 테스트 또는 검증 결과를 기록한다.
- [ ] 필요한 경우 문서 또는 구현 계획을 작성하거나 갱신한다.
- [ ] API, DB, 화면 계약 변경이 있다면 영향 범위를 확인한다.
