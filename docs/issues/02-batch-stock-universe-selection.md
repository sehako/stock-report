## 개요

Python 배치에서 KRX 종목 목록을 조회하고 거래량 기준 상위 200개 종목을 현재 조회 대상으로 갱신한다.

## 작업 범위

- [ ] `FinanceDataReader.StockListing("KRX")`로 KRX 종목 목록을 조회한다.
- [ ] 거래량 기준 상위 200개 종목을 선정한다.
- [ ] 선정 종목을 `stock` 테이블에 upsert한다.
- [ ] 현재 상위 200개 종목은 `tracked = true`로 갱신한다.
- [ ] 이전에 저장되었지만 현재 상위 200개에서 제외된 종목은 `tracked = false`로 갱신한다.
- [ ] ETF, ETN, 우선주 등 비보통주를 별도로 제외하지 않는다.

## 검증 흐름

- [ ] 기본 배치를 실행했을 때 `tracked = true` 종목이 거래량 상위 200개로 저장되는지 확인한다.

## 제외 범위

- 종목 일봉 가격 수집은 제외한다.
- 지수 일봉 가격 수집은 제외한다.
- 지정 재수집 옵션은 제외한다.
- Spring Boot 조회 API 구현은 제외한다.

## 완료 기준

- [ ] 구현 범위가 이슈 목적을 벗어나지 않는다.
- [ ] 관련 테스트 또는 검증 결과를 기록한다.
- [ ] 필요한 경우 문서 또는 구현 계획을 작성하거나 갱신한다.
- [ ] API, DB, 화면 계약 변경이 있다면 영향 범위를 확인한다.
