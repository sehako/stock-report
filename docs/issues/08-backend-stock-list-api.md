## 개요

Spring Boot 백엔드에서 거래량 상위 200개 조회 대상 종목 목록을 검색, 필터, 정렬, 페이지네이션 조건으로 조회하는 API를 제공한다.

## 작업 범위

- [ ] `GET /api/stocks?market=&keyword=&sort=&page=&size=` 엔드포인트를 구현한다.
- [ ] `tracked = true` 종목만 목록 조회 대상으로 제한한다.
- [ ] 종목명 또는 종목코드 `keyword` 검색을 지원한다.
- [ ] 시장 구분 `market` 필터를 지원한다.
- [ ] 최신 종가, 등락률, 거래량 기준 정렬을 지원한다.
- [ ] `page`, `size` 기반 페이지네이션을 지원한다.
- [ ] 응답에 종목코드, 종목명, 시장 구분, 최신 종가, 등락률, 거래량, 마지막 거래일을 포함한다.

## 검증 흐름

- [ ] 검색어, 시장 필터, 거래량 정렬, 페이지네이션을 함께 지정했을 때 조건에 맞는 tracked 종목 페이지만 반환되는지 테스트한다.

## 제외 범위

- 종목 상세 조회 API는 제외한다.
- 종목 차트 조회 API는 제외한다.
- Python 배치 구현은 제외한다.
- React 화면 구현은 제외한다.

## 완료 기준

- [ ] 구현 범위가 이슈 목적을 벗어나지 않는다.
- [ ] 관련 테스트 또는 검증 결과를 기록한다.
- [ ] 필요한 경우 문서 또는 구현 계획을 작성하거나 갱신한다.
- [ ] API, DB, 화면 계약 변경이 있다면 영향 범위를 확인한다.
