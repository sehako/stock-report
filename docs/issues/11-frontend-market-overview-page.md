## 개요

React 프론트엔드에서 코스피와 코스닥 최신 수치와 기간별 차트를 조회하는 시장 개요 화면을 구현한다.

## 작업 범위

- [ ] 시장 개요 라우팅 페이지를 구성한다.
- [ ] 지수 최신 수치 조회 API를 호출한다.
- [ ] 코스피 최신 수치 카드를 표시한다.
- [ ] 코스닥 최신 수치 카드를 표시한다.
- [ ] 전일 대비 값과 등락률을 표시한다.
- [ ] 지수별 1개월, 3개월, 1년 차트 기간 선택을 제공한다.
- [ ] 선택한 기간에 맞춰 지수 차트 조회 API를 호출한다.
- [ ] 로딩, 빈 상태, 오류 상태를 화면 영역별로 표시한다.
- [ ] `docs/architecture/frontend.md`의 기능 중심 구조를 따른다.

## 검증 흐름

- [ ] 시장 개요 화면에서 지수 기간을 변경했을 때 선택한 기간의 차트 데이터가 다시 조회되고 화면에 반영되는지 테스트한다.

## 제외 범위

- 종목 목록 화면은 제외한다.
- 종목 상세 화면은 제외한다.
- Spring Boot API 구현은 제외한다.
- Python 배치 구현은 제외한다.

## 완료 기준

- [ ] 구현 범위가 이슈 목적을 벗어나지 않는다.
- [ ] 관련 테스트 또는 검증 결과를 기록한다.
- [ ] 필요한 경우 문서 또는 구현 계획을 작성하거나 갱신한다.
- [ ] API, DB, 화면 계약 변경이 있다면 영향 범위를 확인한다.
