## 개요

React 프론트엔드에서 단일 종목의 기본 정보, 최신 일봉, 기간별 차트를 조회하는 종목 상세 화면을 구현한다.

## 작업 범위

- [ ] 종목 상세 라우팅 페이지를 구성한다.
- [ ] 종목 상세 조회 API를 호출한다.
- [ ] 종목 기본 정보를 표시한다.
- [ ] 최신 시가, 고가, 저가, 종가, 거래량, 등락률을 표시한다.
- [ ] 1개월, 3개월, 1년 차트 기간 선택을 제공한다.
- [ ] 선택한 기간에 맞춰 종목 차트 조회 API를 호출한다.
- [ ] 존재하지 않는 종목 또는 API 오류 상태를 표시한다.
- [ ] 차트 데이터가 없는 경우 빈 상태를 표시한다.

## 검증 흐름

- [ ] 종목 상세 화면에서 차트 기간을 변경했을 때 선택한 기간의 종목 일봉 데이터가 다시 조회되고 화면에 반영되는지 테스트한다.

## 제외 범위

- 시장 개요 화면은 제외한다.
- 종목 목록 화면은 제외한다.
- Spring Boot API 구현은 제외한다.
- Python 배치 구현은 제외한다.

## 완료 기준

- [ ] 구현 범위가 이슈 목적을 벗어나지 않는다.
- [ ] 관련 테스트 또는 검증 결과를 기록한다.
- [ ] 필요한 경우 문서 또는 구현 계획을 작성하거나 갱신한다.
- [ ] API, DB, 화면 계약 변경이 있다면 영향 범위를 확인한다.
