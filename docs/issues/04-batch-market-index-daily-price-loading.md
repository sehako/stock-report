## 개요

Python 배치에서 코스피와 코스닥 지수의 일봉 가격 데이터를 수집해 `market_index_price`에 저장한다.

## 작업 범위

- [ ] 내부 지수 코드 `KOSPI`, `KOSDAQ`과 FDR 지수 심볼 매핑을 정의한다.
- [ ] 지수별 마지막 적재일을 기준으로 추가 수집 시작일을 계산한다.
- [ ] 최초 적재 대상 지수는 조회 가능한 최초 일봉부터 최신 일봉까지 수집한다.
- [ ] 이후 실행 대상 지수는 마지막 적재일 이후 일봉만 수집한다.
- [ ] 수집한 시가, 고가, 저가, 종가, 거래량, 등락률을 `market_index_price`에 upsert한다.
- [ ] 지수별 수집 성공, 스킵, 실패 결과를 로그로 남긴다.

## 검증 흐름

- [ ] 같은 지수 일봉 적재 배치를 반복 실행해도 `index_code`, `trade_date` 기준 중복 row가 생기지 않는지 확인한다.

## 제외 범위

- 종목 메타데이터 갱신은 제외한다.
- 종목 일봉 가격 수집은 제외한다.
- 지정 재수집 옵션은 제외한다.
- Spring Boot 조회 API 구현은 제외한다.

## 완료 기준

- [ ] 구현 범위가 이슈 목적을 벗어나지 않는다.
- [ ] 관련 테스트 또는 검증 결과를 기록한다.
- [ ] 필요한 경우 문서 또는 구현 계획을 작성하거나 갱신한다.
- [ ] API, DB, 화면 계약 변경이 있다면 영향 범위를 확인한다.
