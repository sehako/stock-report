## 개요

Spring Boot 백엔드에서 단일 종목의 기본 정보와 최신 일봉 데이터를 조회하는 API를 제공한다.

## 작업 범위

- [ ] `GET /api/stocks/{stockCode}` 엔드포인트를 구현한다.
- [ ] 종목코드로 `stock` 메타데이터를 조회한다.
- [ ] 해당 종목의 최신 일봉 데이터를 함께 조회한다.
- [ ] 응답에 종목 기본 정보와 최신 시가, 고가, 저가, 종가, 거래량, 등락률, 거래일을 포함한다.
- [ ] 존재하지 않는 종목코드는 404로 처리한다.
- [ ] 일봉 데이터가 아직 없는 종목은 명확한 결측 상태로 응답한다.

## 검증 흐름

- [ ] 존재하는 종목코드 요청 시 종목 기본 정보와 최신 일봉 데이터가 함께 반환되는지 테스트한다.

## 제외 범위

- 종목 목록 조회 API는 제외한다.
- 종목 차트 조회 API는 제외한다.
- Python 배치 구현은 제외한다.
- React 화면 구현은 제외한다.

## 완료 기준

- [ ] 구현 범위가 이슈 목적을 벗어나지 않는다.
- [ ] 관련 테스트 또는 검증 결과를 기록한다.
- [ ] 필요한 경우 문서 또는 구현 계획을 작성하거나 갱신한다.
- [ ] API, DB, 화면 계약 변경이 있다면 영향 범위를 확인한다.
