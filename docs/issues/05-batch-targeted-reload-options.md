## 개요

Python 배치에 특정 종목, 특정 지수, 특정 기간만 재수집할 수 있는 실행 옵션을 추가한다.

## 작업 범위

- [ ] `--stock-code` 옵션을 지원한다.
- [ ] `--index-code` 옵션을 지원한다.
- [ ] `--start-date`, `--end-date` 기간 옵션을 지원한다.
- [ ] 종목 코드와 기간 조합으로 종목 일봉을 재수집한다.
- [ ] 지수 코드와 기간 조합으로 지수 일봉을 재수집한다.
- [ ] 옵션을 지정하지 않으면 기본 실행 흐름으로 동작하게 유지한다.
- [ ] 지원하지 않는 옵션 조합이나 잘못된 날짜 형식은 명확한 오류로 처리한다.

## 검증 흐름

- [ ] `--stock-code 005930 --start-date 2024-01-01 --end-date 2024-01-31` 실행 시 지정 종목과 지정 기간 데이터만 upsert되는지 확인한다.

## 제외 범위

- 배치 실행 이력 테이블 저장은 제외한다.
- 관리자 화면 또는 배치 실패 관리 화면은 제외한다.
- Spring Boot 조회 API 구현은 제외한다.

## 완료 기준

- [ ] 구현 범위가 이슈 목적을 벗어나지 않는다.
- [ ] 관련 테스트 또는 검증 결과를 기록한다.
- [ ] 필요한 경우 문서 또는 구현 계획을 작성하거나 갱신한다.
- [ ] API, DB, 화면 계약 변경이 있다면 영향 범위를 확인한다.
