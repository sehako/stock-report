# Stock Reports Lab

한국 주식시장의 확정된 일별 데이터를 정리하여 투자자가 시장, 업종, 종목의 흐름을 탐색할 수 있게 하는 리포트 서비스이다.

## Language

**장 마감 리포트**:
해당 거래일의 KRX 정규시장 종가를 기준으로 생성하며, 수집 데이터가 검증된 뒤 공개되는 일별 리포트이다.
_Avoid_: 실시간 리포트, 15:40 리포트

**생성 지연**:
당일 핵심 데이터가 완전성 검증을 통과하지 못해 **장 마감 리포트**가 공개되지 않은 상태이다. 이 상태에서는 불완전한 당일 리포트 대신 직전 리포트를 명확한 기준일과 함께 제공한다.
_Avoid_: 부분 성공, 임시 리포트

**데이터 준비 중**:
선정된 **분석 종목** 하나의 최신 일봉을 가져오기 위해 허용된 재시도를 진행하고 있는 상태이다. 다른 분석 종목의 처리와 **장 마감 리포트** 공개를 막지 않는다.
_Avoid_: 생성 지연, 데이터 갱신 실패

**데이터 갱신 실패**:
선정된 **분석 종목** 하나의 최신 일봉을 허용된 재시도 안에 가져오지 못해 해당 종목의 최신 분석 결과를 공개할 수 없는 상태이다. 다른 분석 종목의 처리와 **장 마감 리포트** 공개를 막지 않으며, 시장·업종 집계에는 실제 반영 종목 수를 함께 표시한다.
_Avoid_: 생성 지연, 데이터 준비 중, 분석 데이터 부족

**당일 거래 없음**:
해당 거래일에 새 일봉이 없지만 이전 거래일까지 기술지표 계산에 충분한 데이터가 있는 **분석 종목**의 상태이다. 마지막 실제 거래일을 기준일로 명시하여 기존 분석은 제공하되, 해당 거래일의 신규 신호에는 포함하지 않는다.
_Avoid_: 데이터 갱신 실패, 분석 데이터 부족

**골든크로스 신호**:
**분석 종목**의 최근 세 거래 일봉 안에서 Stoch MACD의 K선이 D선을 아래에서 위로 돌파한 사건이다. 달력 기준 최근 3일을 의미하지 않으며, **당일 거래 없음** 상태의 종목에는 해당 거래일의 신규 신호를 만들지 않는다.
_Avoid_: 상승 보장, 매수 추천, 최근 3일

**종목 상세 보조 지표**:
**분석 종목**의 상세 화면에서 사용자가 가격 흐름과 **골든크로스 신호**의 계산 맥락을 함께 살펴볼 수 있게 제공하는 MACD, Stoch MACD, 이동평균 지표이다. **골든크로스 신호** 외의 별도 신호나 투자 전략을 의미하지 않는다.
_Avoid_: 추가 신호, 전략 지표, 매수 조건

**분석 중**:
검증된 계산 결과를 바탕으로 AI 서술을 생성하거나 재시도하고 있는 상태이다. 이 상태에서도 검증된 수치, 차트 및 신호는 제공한다.
_Avoid_: 생성 지연

**분석 생성 지연**:
AI 서술 생성이 허용된 재시도 횟수 안에 완료되지 않은 상태이다. **생성 지연**과 달리 검증된 수치, 차트 및 신호는 제공한다.
_Avoid_: 생성 지연, 분석 중

**시장 AI 요약**:
해당 거래일의 활성 **장 마감 리포트**를 입력으로 생성하는 거래일별 AI 서술이다. 금융 데이터 리비전 이력이 아니라 거래일별 최신 요약 상태를 의미한다.
_Avoid_: 리비전별 AI 요약, AI 리비전

**분석 데이터 부족**:
선정된 **분석 종목**의 일봉이 기술지표 계산에 필요한 110개보다 적어 해당 종목의 MACD 및 Stochastic 신호를 계산할 수 없는 상태이다. 이 상태에서도 검증된 현재가와 거래량은 제공하며 **장 마감 리포트** 전체의 공개를 막지 않는다.
_Avoid_: 생성 지연, 분석 중, 분석 실패

**분석 종목**:
해당 거래일의 KRX 종목 목록에서 종가가 1,000원 이상인 종목을 거래량 내림차순으로 정렬해 선정한 상위 200개 종목이다. 매 거래일의 거래량에 따라 구성 종목이 달라질 수 있다.
_Avoid_: 전종목, 시장 전체

**선정 기준 거래량**:
해당 거래일의 KRX 종목 목록에서 **분석 종목**을 선정할 때 사용한 거래량 값이다. 이후 최신 거래량이 달라져도 과거 리포트의 목록과 정렬을 재현하기 위해 리비전 스냅샷에 보존한다.
_Avoid_: 최신 거래량, 누적 거래량

**업종**:
FinanceDataReader의 KRX 설명 목록에서 제공하는 거래소 업종 분류이다. 별도로 만든 11개 섹터나 여러 종목을 임의로 묶은 테마를 의미하지 않는다.
_Avoid_: 섹터, 산업, 테마

**관심종목**:
사용자가 현재 이용 중인 기기에서 다시 확인하기 위해 선택한 **분석 종목**이다. 사용자 계정에 귀속되지 않으며 다른 기기와 동기화되지 않는다.
_Avoid_: 포트폴리오, 보유종목

## Example dialogue

개발자: 오늘 장 마감 리포트의 기준 가격은 시간외 거래 가격인가요?

도메인 전문가: 아니요. KRX 정규시장 종가가 기준이며, 데이터 검증이 끝난 뒤 공개합니다.

개발자: 19시까지 당일 데이터가 완성되지 않으면 일부 내용만 먼저 공개할까요?

도메인 전문가: 아니요. 생성 지연으로 표시하고 직전 장 마감 리포트를 유지합니다.

개발자: 한 종목의 일봉 수집만 실패해도 전체 장 마감 리포트를 생성 지연으로 표시하나요?

도메인 전문가: 아니요. 재시도 중에는 해당 종목만 데이터 준비 중으로 표시하고, 재시도가 끝나면 데이터 갱신 실패로 표시합니다.

개발자: 오늘 거래가 없어서 새 일봉이 없는 종목도 데이터 갱신 실패인가요?

도메인 전문가: 아니요. 데이터가 충분하면 당일 거래 없음으로 표시하고 마지막 실제 거래일 기준 분석을 제공합니다.

개발자: 최근 주말을 포함한 3일 동안 교차가 없으면 골든크로스 신호가 아닌가요?

도메인 전문가: 달력 3일이 아니라 최근 세 거래 일봉 안에서 발생한 교차를 의미합니다.

개발자: 데이터 계산은 끝났지만 AI 요약을 재시도하고 있으면 무엇을 보여 주나요?

도메인 전문가: 검증된 결과는 공개하고 AI 영역은 분석 중으로 표시합니다. 재시도가 모두 실패하면 분석 생성 지연으로 바뀝니다.

개발자: 신규 상장 종목의 일봉이 110개보다 적으면 장 마감 리포트를 생성 지연으로 표시하나요?

도메인 전문가: 아니요. 해당 종목만 분석 데이터 부족으로 표시하고 검증된 현재가와 거래량은 제공합니다.

개발자: 전체 종목 표에는 KRX의 모든 종목이 들어가나요?

도메인 전문가: 아니요. 분석 종목은 종가 1,000원 이상인 종목 중 거래량 상위 200개만 의미합니다.

개발자: 2차전지를 하나의 업종으로 표시할까요?

도메인 전문가: 아니요. 업종은 KRX 설명 목록에서 제공한 분류를 그대로 사용합니다.

개발자: 관심종목은 사용자의 다른 기기에도 자동으로 나타나나요?

도메인 전문가: 아니요. 관심종목은 사용자가 현재 이용 중인 기기에만 속합니다.
